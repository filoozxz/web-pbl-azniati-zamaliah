A. Latar Belakang Kasus

Ny. A, seorang ibu hamil 22 tahun usia kehamilan 30 minggu, datang dengan keluhan sakit kepala, bengkak kaki, cepat lelah, dan pandangan kabur. Tekanan darah meningkat, proteinuria positif, dan riwayat ibu dengan preeklamsia.

B. Data Pengkajian

1. Data Subjektif

  • “Kepala saya nyut-nyutan…”
  • “Kaki saya bengkak…”
  • "Kadang pandangan saya kabur…”

2. Data Objektif

  • TD: 148/92 mmHg
  • Protein urin: (+1)
  • Edema grade 1–2
  • Nadi: 96x/menit

C. Analisis (Model Adaptasi Roy)

Mode Fisiologis: Hipertensi, proteinuria, edema.

Mode Konsep Diri: Ibu cemas tentang kehamilannya.

Mode Fungsi Peran: Ibu baru, minim pengetahuan risiko.

Mode Interdependensi: Dukungan ada, tapi kurang edukasi.

D. Diagnosa Keperawatan

  1. Risiko tinggi preeklamsia terkait hipertensi.
  2. Kurang pengetahuan terkait tanda bahaya kehamilan.
  3. Cemas terkait kondisi kesehatan.

E. Intervensi Keperawatan

  • Monitoring tekanan darah dan urine.
  • Edukasi istirahat miring kiri.
  • Kelas ibu hamil & dukungan keluarga.
  • Kolaborasi dokter untuk terapi farmakologi.

F. Pertanyaan Diskusi PBL

  1. Diagnosis awal apa yang ditegakkan?
  2. Apa faktor risiko Ny. A?
  3. Bagaimana analisis Roy Model?
  4. Intervensi prioritas?
  5. Edukasi penting apa?
  6. Potensi komplikasi?
  7. Bagaimana web edukasi membantu?

G. Kesimpulan

Deteksi dini preeklamsia sangat penting karena dapat membahayakan ibu dan janin. Model Adaptasi Roy membantu menilai pasien secara holistik dan menentukan intervensi yang tepat. Web edukasi dapat menjadi media efektif untuk pemahaman ibu dan keluarga.
